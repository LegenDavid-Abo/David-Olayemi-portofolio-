David Olayemi – Portfolio Website

Modern single-page portfolio built with HTML, Tailwind CSS (CDN), and Vanilla JavaScript.

Overview

This website showcases:
	•	Professional summary
	•	Technical skills
	•	Work experience (Nigeria, Dubai, Germany)
	•	AI & Machine Learning projects
	•	Awards and certifications
	•	Contact form
	•	Interactive AI chatbot assistant

The design uses glassmorphism, gradient effects, scroll animations, and a responsive layout.

Tech Stack
	•	HTML5
	•	Tailwind CSS
	•	JavaScript
	•	Lucide Icons
	•	Google Fonts

No backend required. Fully static and deployable on any hosting platform (GitHub Pages, Netlify, Vercel, etc.).

Features
	•	Animated skill bars
	•	Scroll reveal animations
	•	Interactive chatbot with predefined responses
	•	Responsive design
	•	Modern UI/UX styling
